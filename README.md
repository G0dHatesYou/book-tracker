# BookStreak 📚

A social reading tracker with competitive streaks for two.

> **This is a pet project** built to explore code generation capabilities using an agentic AI system powered by Claude (Anthropic). All code — HTML, CSS, and JavaScript — was generated through a conversation with AI without manual coding.

## Features

- Name-based registration (no password required)
- Daily reading page tracking
- Choose a rival from the list of registered users
- Streaks and overall reading statistics
- 28-day activity heatmap
- Real-time data sync between users

## Stack

- Vanilla HTML + CSS + JavaScript, no frameworks
- [JSONBin.io](https://jsonbin.io) as a cloud data store
- Hosted on [GitHub Pages](https://pages.github.com)

## Structure

```
index.html   — markup
style.css    — styles
app.js       — logic and API calls
```

## Running

Open `index.html` in a browser, or deploy all three files to GitHub Pages.
